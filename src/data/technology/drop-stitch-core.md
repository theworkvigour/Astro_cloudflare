Drop-stitch core construction uses thousands of polyester threads connecting the top and bottom PVC layers. When inflated to 12-15 PSI, these threads create a rigid, flat structure that performs like a hard-shell board.

This technology enables Wavefella boards to be packed into a backpack while providing the rigidity needed for stable paddling, standing, and maneuvering.
